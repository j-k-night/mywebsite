Introduction to Web Development

Prog 109

Introduces fundamental programming techniques using current web development software. Students design procedures and write computer instructions to solve business problems, learn procedural programming, develop graphical user interfaces for the web and work with events and objects. Prerequisite: Placement by assessment into ENGL& 101, or completion of ENGL 092 or 093 with a C or better.

5 credits

Maricel Medina-Mora

My reflection on this course

The most important thing I gained from this course is experience using Javascript to manipulate web pages and validate forms. Prior to this class I was familiar with HTML, CSS, and basic Javascript programming - such as variables, loops, etc. - but during this class I gained experience making dynamic web pages that, for example, respond to user input by adding new elements to the page.
